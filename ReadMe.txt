1、項目個構成
    a、項目基礎類庫->server-common 對底層網絡通訊框架和常用函數進行封裝和集成；
    b、新手卡服務器->server-cdk 主要提供新手卡的生成及使用接口;
    c、遊戲服務器->server-game 主要提供遊戲相關業務功能模塊的處理;
    d、合服工具->server-merge 主要提供相關大區遊戲賬號合併功能;
    e、支付服務器->server-pay 主要提供遊戲支付訂單的驗證及發貨功能;
    f、壓力測試工具->server-test

2、開發環境
    1、安裝 jdk1.8;
    2、eclipse開發環境;
    3、maven運行環境配置;
